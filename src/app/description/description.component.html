This is our final project in ICBV course at Ben Gurion university of the Negev in Israel.
We implemented a Fourier transform based algorithm which generates a drawing animation of an image edges using a set of base wave images, each represented by a circle.
You can either upload an image of your choice or draw your personal drawing, and our algorithm will generate its animation for you.
We hope you're going to enjoy it and admire the magic of fourier transform thanks to that.
